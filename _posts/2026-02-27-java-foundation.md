> 说明：本文基于 CSDN、博客园等公开技术文章的共性知识路径整理，适合做阶段化学习计划。

## 学习阶段总览

### 阶段一：语法与面向对象
- 学习重点：变量、流程控制、类与对象、集合、异常、泛型。
- 阶段产出：完成基础语法刷题 + 一个控制台管理系统。

### 阶段二：并发与JVM基础
- 学习重点：线程模型、线程池、内存模型、类加载与垃圾回收。
- 阶段产出：能够分析常见并发问题并完成 JVM 参数调优演练。

### 阶段三：工程实践
- 学习重点：Maven/Gradle、单元测试、日志规范、代码重构。
- 阶段产出：输出一个可部署的 Java Web 小项目。

## 推荐学习方式
- 每个阶段按“概念 -> Demo -> 项目复盘”推进。
- 每周至少安排 1 次错题/故障复盘，沉淀为个人知识库。
- 阶段结束输出一篇总结文档，持续完善跳转索引。
